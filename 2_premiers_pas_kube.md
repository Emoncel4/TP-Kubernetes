# TP 2 Premiers pas sur Kubernetes

## Objectifs du TP
- Déployer une application sur Kubernetes 
- Comprendre la syntaxe des objets Kubernetes
- Utiliser l'outil kubectl 


Livrable : 
- Manifest kubernetes pour déployer les applications 
- Un compte rendu avec les difficultés rencontrées, et les résultats de chaque exercice. 

## Prérequis

- Avoir un compte SSPCloud 

## Déploiement d'une API Python 

Comme présenté dans le cours, l'objectif ici est de déployer 3 replicas de notre application "Hello World".

Vous pouvez récupérer l'image Docker précédemment utilisée et envoyé sur Docker Hub. 

Bonne chance !

PS : Votre application n'est pas exposée d'un point de vue réseau pour le moment, il vous faudra trouver un moyen pour tester cette dernière ! Si ChatGPT (ou autres), vous indique de créer un service, ce n'est pas la solution attendue. 

Dans le compte rendu de cet exercice, il faudra inclure les captures d'écran des commandes permettant :
- d'afficher la liste des pods
- d'afficher la liste des deployments
- de regarder les logs de l'application avant et après test
- de lister les dépendances de l'application 
- de récupérer une description complète de votre application.

Vous commenterez brièvement chaque élément afin de montrer votre bonne compréhension.
